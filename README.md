Airways Review Analytics Dashboard ~ Data visualization dashboard for British Airways customer review analysis

Airways Review Analytics Dashboard is an interactive Tableau project that transforms British Airways customer review data into meaningful business insights through dynamic visualizations and dashboards. The project analyzes customer satisfaction across multiple dimensions, including countries, travel periods, aircraft types, and various service quality metrics such as seat comfort, cabin staff service, food, entertainment, and overall rating. By leveraging Tableau's interactive features like parameters, calculated fields, filters, maps, and charts, the dashboard enables users to explore trends, compare performance, and uncover patterns that can support data-driven decision-making and improve customer experience.

Features:

* Interactive world map showing country-wise customer ratings
* Dynamic metric selection using Tableau parameters
* Custom calculated fields for flexible analysis
* Monthly trend analysis using line charts
* Aircraft-wise comparison using dual bar charts
* Interactive filters for customized data exploration
* Responsive dashboard with easy-to-navigate visualizations
* Clean dashboard layout designed for business insights

How to Use:

1. Clone or download the repository.
2. Open the Tableau Workbook (.twb or .twbx) using Tableau Desktop or Tableau Public.
3. If required, reconnect the workbook to the provided dataset.
4. Explore the dashboard using the interactive filters and parameter controls.
5. Select different customer rating metrics to compare performance across countries, months, and aircraft types.
6. Analyze trends and insights through the interactive visualizations.
